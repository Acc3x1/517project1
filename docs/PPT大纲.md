- 用Java+JavaFX开发仿真系统

### 第6页：核心算法 - 温度仿真引擎

**5通道温度**：
- TF1（炉温1）: 升温→稳定750°C
- TF2（炉温2）: 同TF1，独立噪声
- TS（表面温度）: 指数跟随炉温×0.95
- TC（中心温度）: 指数跟随炉温×0.85
- TCal（校准温度）: TF1 + 噪声×2

**算法公式**：

升温阶段：
```
TF1 += HeatingRate × 0.8 + Random(-1,1) × 0.5
```

稳定阶段：
```
TF1 = 750 + Random(-1,1) × 0.5
stableCount++，当>3时IsStable=true
```

记录阶段：
```
TS += (TF1 × 0.95 - TS) × 0.02 + noise
TC += (TF1 × 0.85 - TC) × 0.01 + noise
```

### 第7页：状态机设计

**5状态流转图**：
```
     ┌─────────┐
     │  IDLE   │ ←── 停止升温
     │ (空闲)  │
     └────┬────┘
          │ 开始升温
          ↓
     ┌─────────┐
     │PREPARING│ ←── 温度跌出范围
     │(升温中) │
     └────┬────┘
          │ TF1≥747°C 且 stableCount>3
          ↓
     ┌─────────┐
     │  READY  │
     │ (就绪)  │
     └────┬────┘
          │ 开始记录
          ↓
     ┌─────────┐
     │RECORDING│
     │(记录中) │
     └────┬────┘
          │ 60分钟/手动停止
          ↓
     ┌─────────┐
     │COMPLETE │ →── 保存后继续升温
     │ (完成)  │
     └─────────┘
```
